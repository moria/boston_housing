编程练习3：定义衡量标准Â¶

如果不能对模型的训练和测试的表现进行量化地评估，我们就很难衡量模型的好坏。通常我们会定义一些衡量标准，这些标准可以通过对某些误差或者拟合程度的计算来得到。在这个项目中，你将通过运算决定系数 R2 来量化模型的表现。模型的决定系数是回归分析中十分常用的统计信息，经常被当作衡量模型预测能力好坏的标准。

R2的数值范围从0至1，表示目标变量的预测值和实际值之间的相关程度平方的百分比。一个模型的R2 值为0还不如直接用平均值来预测效果好；而一个R2 值为1的模型则可以对目标变量进行完美的预测。从0至1之间的数值，则表示该模型中目标变量中有百分之多少能够用特征来解释。模型也可能出现负值的R2，这种情况下模型所做预测有时会比直接计算目标变量的平均值差很多。

在下方代码的 performance_metric 函数中，你要实现：

  • 使用 sklearn.metrics 中的 r2_score 来计算 y_true 和 y_predict的R2值，作为对其表现的评判。
  • 将他们的表现评分储存到score变量中。

或

  • (可选) 不使用任何外部库，参考决定系数的定义进行计算，这也可以帮助你更好的理解决定系数在什么情况下等于0或等于1。

def performance_metric(y_true, y_predict):
    """计算并返回预测值相比于预测值的分数"""
    
    score = r2_score(y_true, y_predict)

    return score

问题 8 - 回答：Â¶

k折交叉验证法即将数据集且分为k份，任选一份为验证集，其余为训练集，进行k次训练和验证，对预测评分求均值，从而能够在同一份数据上进行多次训练和验证，得到更稳定的预测评分，避免数据集划分的偶然性导致评分的波动性。

GridSearchCV将不同参数的组合在不同的K个（训练-验证）集上进行迭代训练-预测，score_func取值最大的参数组合即认为是最为稳定和高性能的参数组合

cvresults可以看出，GridSearchCV函数通过在不同的训练-验证集（此处为3个）组合上进行不同参数组合（此处为max_depth0-10)的评分，最终按照std评分较高的那一个参数组合返回确认的最优参数

网格搜索如果不使用交叉验证会使得返回的参数组合训练产生的模型稳定性不足。交叉验证使得GridSearchCV函数在K个不同的训练验证集上得到了一个最优的参数组合，其稳定性比单单从一个训练验证组合得到的最优参数组合理论上要高。例如，如果恰巧我们使用kfold中split2这个训练验证集进行训练和验证，并进行评分后选择最优的，我们会选择max_depth=6(评分为-0.19,要高于max_depth=4的-0.52）作为最优参数，这样，无论从可视化结果还是从实际得到的k个训练集的标准差数据来看，都不是一个最优的选择。

def cross_val_score(estimator, X, y, scoring = performance_metric, cv=3):
    """ 返回每组交叉验证的模型分数的数组 """
    test_len = len(X)/cv
    train_len = len(X) - test_len
    i = cv
    scores = []
    while i > 0:
        start = len(X) - i * test_len
        if start < 0:
            start = 0
        test_x = X[start:start + test_len]
        test_y = y[start:start + test_len]
        if start > 0 :
            train_x = pd.concat([X[0:start], X[(start+test_len):]])
            train_y = pd.concat([y[0:start], y[(start+test_len):]])
        else:
            train_x = X[(start+test_len):]
            train_y = y[(start+test_len):]
                            
        estimator.fit(train_x, train_y)
        predict_y = estimator.predict(test_x)
        score = scoring(test_y, predict_y)               
        scores.append(score)
        i = i - 1
        
    return scores
        

def fit_model2(X, y):
    """ 基于输入数据 [X,y]，利于网格搜索找到最优的决策树模型"""    
    #最优交叉验证分数对应的最优模型
    best_estimator = None
    max_tot = 0
    
    for depth in range(1, 11):
        clf = DecisionTreeRegressor(max_depth=depth)
        scores = cross_val_score(clf, X, y)
        tot = 0
        for i in scores:
            tot = tot + i
            if tot >= max_tot:

                best_estimator = clf
